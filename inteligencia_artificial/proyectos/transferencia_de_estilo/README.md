# Transferencia de estilo entre imágenes 
Se muestra un proyecto el cual toma 2 imágenes que uno quiera y transfiere el estilo o características de 
una de las imágenes a la otra según el usuario quiera utilizando una *red neuronal VGG-19*, el objetivo es 
imprlmentar este tipo de arquitectura para comprender mejor como funciona de una forma muy visual.
